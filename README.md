# Heartful Hound Website

## Overview

This is a professional website built for **Heartful Hound**, a dog behaviour and training business based in South Africa. The website is designed to present services in a clear, approachable, and trustworthy way while making it easy for potential clients to get in touch.

The focus of this project is to create a calm, friendly, and professional online presence that reflects the values of animal care, behaviour understanding, and effective dog training.

---

## Live Website

Visit the live site:
https://www.heartfulhound.co.za

---

## Features

* Clean, modern, and calming design
* Fully responsive layout (mobile, tablet, desktop)
* Services section for dog training and behaviour support
* Contact information for client enquiries
* User-friendly navigation
* Fast-loading and lightweight structure
* SEO-friendly layout for better visibility

---

## Technologies Used

* HTML5
* CSS3
* JavaScript

---

## Purpose of This Website

The purpose of this website is to help **Heartful Hound** attract and inform clients by clearly presenting services related to dog behaviour and training. It is designed to build trust, improve online visibility, and make it easy for pet owners to understand and access professional support.
